# 📄 PODSTRONA: Projekty (Projects)

Komponenty używane na stronach portfolio projektów.

## 📍 Lokalizacja stron
- Lista: `src/app/[lang]/projekty/page.tsx`
- Szczegóły: `src/app/[lang]/projekty/[slug]/page.tsx`

## 🎯 Typy projektów
- Enterprise LLM deployments
- Computer Vision systems
- Agentic AI implementations
- MLOps infrastructure
- Custom ML models

## 🔧 Komponenty do dodania
- `ProjectCard.tsx` - Karta projektu z obrazkiem
- `ProjectGrid.tsx` - Grid z projektami
- `ProjectHero.tsx` - Hero dla pojedynczego projektu
- `ProjectDetails.tsx` - Szczegóły projektu
- `ProjectResults.tsx` - Wyniki i metryki
- `ProjectTechnology.tsx` - Użyte technologie
- `RelatedProjects.tsx` - Podobne projekty

## 📝 Zasady
- Komponenty używane TYLKO na stronach projektów
- Współdzielone componenty w `src/components/shared/`
