# 📄 PODSTRONA: O brillQ (About)

Komponenty używane na stronie "O nas" / "Grupa".

## 📍 Lokalizacja strony
`src/app/[lang]/grupa/page.tsx`

## 🎯 Zawartość
- Misja i wizja firmy
- Wartości
- Historia rozwoju (timeline)
- Liczby i statystyki
- Zespół
- Call to action

## 🔧 Komponenty
Obecnie strona używa inline komponentów.
W przyszłości można przenieść tutaj:
- `AboutHeroSection.tsx`
- `MissionVisionSection.tsx`
- `ValuesGrid.tsx`
- `CompanyTimeline.tsx`
- `TeamSection.tsx`
- `StatsSection.tsx`

## 📝 Zasady
- Komponenty używane TYLKO na stronie "O nas"
- Współdzielone komponenty (AnimatedSection, etc.) w `src/components/shared/`
